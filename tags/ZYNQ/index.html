ZYNQ中的GPIO

GPIO 原理

GPIO 通过 MIO 提供 54 路接口，其中 16 路位于 bank 500，剩余位于 bank 501。还通过 EMIO 接口提供从 PL 来的 64 路输入和 128 路输出。GPIO 控制和状态寄存器内存映射在基址 。

GPIO 内部结构

器件引脚的 GPIO 控制

  1. DATA_RO：总是读取 GPIO 引脚的状态，不考虑 GPIO 是输入还是输出，忽略对该寄存器的写操作。

    [!note]+
    如果没有将 MIO 配置为 GPIO，则 DATA_RO 的内容不可读。

  2. DATA：当把 GPIO 配置为输出时，该寄存器用于控制给输出的值，一次性对这 32 位寄存器进行写操作。当读取该寄存器时，返回上次写到 DATA 或 MASK_DATA_[LSW, MSW]的值，不返回当前引脚的值。
  3. DIRM：方向模式，控制 I/O 引脚是输入还是输出， 时禁止输出驱动器。
  4. OEN：输出使能，当 I/O 配置为输出时，控制使能/禁止输出。当 时禁用输出驱动器，禁止时引脚为三态。
